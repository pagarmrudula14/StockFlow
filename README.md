# StockFlow

# 🚀 StockFlow — Production-Ready Full Stack Trading Platform

A modern stock trading platform inspired by Zerodha Kite, engineered using the MERN Stack and deployed across multiple cloud platforms.

### 🌐 Live Applications

**Frontend:** https://stock-flow-bjt31yfh0-mrudula.vercel.app

**Trading Dashboard:** https://stock-flow-8edu.vercel.app

**Backend API:** https://stockflow-3oo6.onrender.com

---

## 📊 Impact & Engineering Scale

✅ Architected and deployed **3 independent production applications**

✅ Built **Full-Stack MERN Architecture** with separate Frontend, Dashboard, and Backend services

✅ Developed **5+ REST APIs** for portfolio, holdings, positions, orders, and authentication

✅ Implemented **OTP-based Authentication Flow** with secure user onboarding

✅ Engineered **6 Core Trading Modules**

* Dashboard
* Holdings
* Positions
* Orders
* Funds
* Profile Management

✅ Integrated **MongoDB Database Layer** with multiple collections and schema relationships

✅ Built **Portfolio Analytics & Interactive Visualizations** using Chart.js

✅ Deployed applications across **2 Cloud Platforms**

* Vercel
* Render

✅ Designed responsive interfaces optimized for desktop and mobile devices
